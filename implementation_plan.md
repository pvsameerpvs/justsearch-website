# Implementation Plan - JustSearch Website

## Overview
This plan outlines the development of the JustSearch UAE website, a premium digital marketing and business listing platform built with Next.js, TypeScript, Tailwind CSS, and Shadcn UI.

## Technology Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Components**: Shadcn UI
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Outfit (Headings), Inter (Body)

## Project Structure
- `src/app/`: Core routing and layout
- `src/components/`: Shared components (Navbar, Footer)
- `src/components/sections/`: Page-specific sections (Hero, About, Services, etc.)
- `src/components/ui/`: Shadcn UI atomic components
- `src/lib/`: Utility functions

## Key Features
1. **Glassmorphism Navbar**: Adaptive sticky navigation with mobile support.
2. **Interactive Hero**: High-impact visual with animated growth metrics.
3. **Advanced Services Grid**: Dedicated cards for Website, SEO/AEO, Social Media, Lead Gen, **Mobile Apps**, and **Custom Software (CRM/ERP)**.
4. **Multi-Tab Pricing**: Organized switchable plans for Websites, SEO, **Software & Apps**, and Directory listings.
5. **Team Profiles**: Individual contact cards for consultants.
6. **Dark Mode Support**: Fully responsive dark theme optimized for premium aesthetics.

## Design System
- **Primary Color**: `#F26A4B` (JustSearch Orange)
- **Secondary Color**: `#FF8A65` (Coral Gradient)
- **Background**: `#FFFFFF` / `#F5F6FA` (Light) | `#1A1D23` (Dark)
- **Text**: `#2F3640` (Headings) | `#555555` (Body)
